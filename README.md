MySQL
========

Simple role for installing MySQL and setting up a db and user.
Note that the database, username and password will all be the same bu default

Requirements
------------

None

Role Variables
--------------

mysql_db_name: example

Dependencies
------------

None

Example Playbook
-------------------------

    - hosts: default
      roles:
        - ShadowKoBolt.mysql
